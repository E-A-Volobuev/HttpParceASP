Парсер.
Руководство по запуску приложения:
Для запуска приложения выберите пункт меню "Парсинг",
в открывшейся вкладке введите название сайта без "www"(пример-yandex.ru).
Для просмотра результатов парсинга, перейдите в главное меню и выберите пункт
"История запросов".
Приятного использования!


Руководство пользователя:
Программа "Парсер" основана на инфраструктуре ASP.NET MVC5.
Для сохранения данных используется база данных MSSQL, для обращения к базе
данных из кода программы используется контекст данных и ENTITY FRAMEWORK.
При выбрасывании ошибки или исключения нажмите F5 и откроется "Журнал ошибок" 
с описанием ошибки или исключения.
